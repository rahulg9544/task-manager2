

## 🛠 Setup Instructions
1.  `composer install`
2.  `cp .env.example .env` (Update DB credentials for MySQL)
3.  `php artisan key:generate`
4.  `php artisan migrate --seed`
5.  `npm install && npm run dev`
6.  task_manager db can directly upload
7.  After run php artisan serve
